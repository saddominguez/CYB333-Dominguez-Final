# CYB333-Dominguez-Final
Password Strength Analyzer

## Libraries

My project uses Python libraries:

**re**
This is used to check the passwords for patterns.
**getpass**
This is used to allow a user to enter a password without displaying it on the screen

## Requirements
- Displays password requirements before input
- Checks password length (minimum 12 characters)
- Requires uppercase and lowercase letters
- Requires at least one number
- Requires at least one special character
- Detects common weak patterns (ex: password, 123)
- Denies weak passwords and provides suggestions
- Provides a password strength score and rating
